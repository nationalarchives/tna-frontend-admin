# TNA Frontend Admin

A set of styles and components designed for National Archives admin interfaces which extends [TNA Frontend](https://github.com/nationalarchives/tna-frontend).

## Quickstart

```sh
# Install dependencies
npm install

# Start Storybook
npm start
```
